# 地推渠道活码系统 使用指南

> 🎯 一套帮助您追踪线下推广效果的二维码管理工具

## 这个系统能做什么？

简单来说，这个系统可以帮您：

1. ✅ 为每个**门店**和**员工**生成专属的推广二维码
2. ✅ 用户扫码后，系统自动**记录是谁带来的客户**
3. ✅ 在后台**查看数据**，知道哪个门店、哪个员工推广效果最好

---

## 🚀 快速上手（5 分钟学会）

### 第一步：配置小程序

进入系统后，首先在「**配置管理**」页面添加您的小程序信息：

| 需要填写的内容 | 说明 | 在哪里找到 |
|--------------|------|-----------|
| 小程序名称 | 给配置起个名字，方便识别 | 自己填写 |
| AppID | 小程序的唯一标识 | 微信公众平台 → 开发 → 开发设置 |
| AppSecret | 小程序的密钥 | 微信公众平台 → 开发 → 开发设置 |
| 落地页路径 | 用户扫码后进入的小程序页面 | 如：`pages/index/index` |

> 💡 **提示**：填写完成后，记得点击「**激活**」按钮，这样生成二维码时才会使用这个配置。

---

### 第二步：创建推广活动（可选）

在「**活动管理**」页面创建活动，可以让扫码页面更好看：

- **活动名称**：比如"春节促销"、"五一活动"
- **主题色**：设置页面的主色调
- **标题文字**：用户扫码后看到的提示语
- **按钮文字**：如"立即进入小程序"
- **Logo 图片**：展示您的品牌标识

> 💡 **提示**：活动是可选的，不创建活动也能生成二维码。

---

### 第三步：生成二维码

这是最重要的一步！在「**二维码管理**」页面：

1. 填写**门店 ID** 和 **门店名称**（如：001、北京朝阳店）
2. 填写**员工 ID** 和 **员工姓名**（如：1001、张三）
3. 如果有活动，可以选择关联的活动
4. 点击「**生成二维码**」

生成后，您可以：
- 👁️ **预览**：查看二维码效果
- ⬇️ **下载**：保存二维码图片，打印分发给员工
- 🗑️ **删除**：不需要的二维码可以删除

---

### 第四步：查看推广数据

在「**扫码看板**」页面，您可以看到：

- 📊 **总扫码次数**：多少人扫过您的二维码
- 👥 **注册人数**：多少人完成了注册
- 🏪 **门店排名**：哪个门店推广效果最好
- 👤 **员工排名**：哪个员工带来客户最多

---

## 📱 用户扫码后会发生什么？

```
用户扫码 → 看到推广页面 → 点击按钮 → 进入小程序 → 系统自动记录来源
```

1. 用户用微信扫描员工的专属二维码
2. 手机上会显示一个推广页面（如果设置了活动，会显示活动样式）
3. 用户点击按钮后，自动跳转到您的小程序
4. 系统会记录这个用户是通过哪个门店、哪个员工带来的

---

## ❓ 常见问题

### Q：我可以给同一个员工生成多个二维码吗？
**A**：可以，每次生成的二维码都是独立的。但建议一个员工使用一个固定的二维码，方便统计。

### Q：二维码生成后可以修改门店信息吗？
**A**：不可以。如果需要修改，请删除原二维码后重新生成。

### Q：如何获取小程序的 AppID 和 AppSecret？
**A**：登录[微信公众平台](https://mp.weixin.qq.com/) → 开发 → 开发管理 → 开发设置，即可查看。

### Q：扫码数据是实时更新的吗？
**A**：是的，用户扫码后数据会立即更新到看板。

### Q：二维码有有效期吗？
**A**：二维码本身没有有效期，只要不删除就一直有效。如果关联了活动，活动结束后扫码页样式可能变为默认样式。
